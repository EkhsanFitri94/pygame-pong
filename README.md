Pong
A classic Pong game clone created with Pygame to learn fundamental 2D game mechanics.

How to Run
1. Make sure you have Python installed.
2. Install the required library by running: pip install -r requirements.txt
3. Run the game: python pong.py

How to Play
- Use the UP and DOWN arrow keys to move your paddle (on the right).
- First to 7 points wins! (The game doesn't stop, but you can see who is ahead).
- Press the red 'X' to close the game.

Skills Demonstrated
- Implementing the core game loop.
- Handling user input (keyboard events).
- Drawing shapes and text to the screen.
- Implementing basic collision detection.
- Managing game state (score).
- Simple AI for the opponent.
